### General Info
***
Restaurant API is a school project to learn build API's using Mongoose and Node.js.
For now, this is only back-end.

Back-end can be tested using Postman. Following routes work.
* xxx/api/getall
* xxx/api/:id
* xxx/api/add
* xxx/api/update/:id
* xxx/api/delete/:id
